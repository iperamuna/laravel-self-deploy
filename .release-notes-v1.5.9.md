# Release Notes - v1.5.9 (2026-02-12)

### 🐞 Bug Fixes

- **Exit Code Detection**: Fixed a critical bug in the `selfdeploy:run` command where systemd unit execution was incorrectly reported as a failure. This was caused by misinterpreting the integer return values from the PHP `exec()` function. The command now reliably detects when a systemd unit has started successfully.
